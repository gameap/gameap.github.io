Quake III Arena — это мультиплеерный шутер от первого лица, разработанный id Software. 
Игра полностью сфокусирована на многопользовательских сражениях, без развитой одиночной кампании.

Игра похожа на Unreal Tournament, геймплей также похож на другие шутеры от первого лица, 
выпущенные в конце 1990-х и начале 2000-х годов.

## Настройка окружения

GameAP обеспечивает поддержку игровых серверов Quake III Arena, их настройку и управление.

* [Установка GameAP на Linux](/ru/install/install_on_linux.html)
* [Установка GameAP на Windows](/ru/install/install_on_windows.html)

### Установка GameAP Daemon

GameAP Daemon представляет собой агент, который отвечает за управление игровыми серверами на выделенных машинах.
Для установки игрового сервера на машине (VDS) необходимо установить GameAP Daemon.

При установке GameAP вы можете выбрать полную установку, включая Daemon (флаг `--with-daemon`).

В панели управления перейдите на страницу **"Администрирование"** -> **"Выделенные серверы"** -> **"Создать"**. 
Появится окно с предложением автоматической установки. Скопируйте код и выполните его на выделенном сервере.

После этого можно приступить к установке сервера Quake III Arena.

## Установка Quake III Arena сервера в GameAP

Откройте раздел **Администрирование** -> **Игровые серверы** -> **Создать**

![](/images/ru/tutorials/quake3/create_form.png)

* В поле "Имя" укажите произвольное название сервера, например "Мой Quake III сервер".
* В поле "Игра" выберите "Quake 3" из выпадающего списка.
* В поле "Модификация" укажите нужную модификацию, по умолчанию установлен `ioquake3`. Также доступен вариант `quake3e`.
* В поле "Выделенный сервер" укажите ноду, на которой будет размещён игровой сервер.
  По умолчанию выбирается первая доступная нода.
* В поле "IP" выберите нужный адрес для вашего сервера, после чего укажите свободный порт
  или оставьте предложенный системой.

## Настройка сервера Quake III Arena

Для изменения конфигурации сервера перейдите в раздел **Сервер**, выберите ваш сервер и нажмите **Управление**.
После этого откройте вкладку **Настройки**.

![](/images/ru/tutorials/quake3/settings.png)

Чтобы изменения вступили в силу, необходимо перезапустить сервер.

### Server Hostname / Имя сервера

Название вашего сервера Quake III, которое будет видно всем игрокам
в списке серверов при поиске.

### Maximum players on server / Максимальное количество игроков на сервере

Предельное число игроков, которые могут одновременно находиться на сервере.
Значение по умолчанию — 16.

### Default map of the server / Карта по умолчанию на сервере

Карта, которая будет автоматически загружаться при запуске сервера.
По умолчанию используется карта `q3dm17`, известная под названием "The Longest Yard".

### Timelimit in minutes / Лимит времени в минутах

Максимальная продолжительность раунда в минутах. Значение по умолчанию — 20 минут.

### Frag limit / Лимит фрагов

Количество фрагов (убийств), при достижении которого раунд завершается.
Значение по умолчанию — 20 фрагов.
