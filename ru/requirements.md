### Composer

Желателен, но не обязателен. Необходим для ручной установки.

#### Установка

##### Linux
```bash
wget https://getcomposer.org/download/1.8.0/composer.phar
cp composer.phar /usr/bin/composer
chmod +x /usr/bin/composer
```

### Git

Желателен, но не обязателен. Необходим для ручной установки.

#### Установка

##### Debian/Ubuntu

```

```
